# FlexClient

**1.21.1 combat + utility client for anarchy.**

Me and my friend byBurak started working on this client almost a year ago. We had to leave the project on its own for a while because life got in the way and we needed to focus on personal stuff, but now we are back.

Lately, we have been seeing people push garbage ratted paid clients, and honestly, we are tired of it. Phobos was a rat, and some things never really change. Same mess, different name.

So we decided it was time to take control of this battle.

Yeah, we know 1.21.1 is not the newest version anymore, but the base I made around 6 months ago made both me and Burak believe in this project again.

So yeah, Flex is coming.

We also have some ideas for people who can’t make their own base and are looking for a free client. We still haven’t fully decided yet, but a free alternative to Flex might happen.

Stay tuned.
