# Security Policy

## Supported Versions

FlexClient is still in development and does not have a public stable release yet.

| Version | Supported |
| ------- | --------- |
| Private development builds | :white_check_mark: |
| Public releases | Coming soon |
| Unofficial builds / reuploads | :x: |

Only builds released by the official FlexClient team are supported. Do not trust random reuploads, cracked builds, or files sent by other people.

## Reporting a Vulnerability

If you find a security issue in FlexClient, please report it privately instead of posting it publicly.

You can report:

* Suspicious behavior
* Possible RAT/backdoor behavior
* Token grabbing or credential-related issues
* Unsafe update/download behavior
* Vulnerabilities in the client, launcher, installer, or website
* Fake builds pretending to be FlexClient

When reporting, please include as much detail as possible:

* What happened
* Which build/version you used
* Where you downloaded it from
* Steps to reproduce the issue
* Screenshots, logs, or proof if available

We will review valid reports as soon as possible. If the report is accepted, we will work on a fix and update the community when needed.

Do not publish exploits, bypasses, or harmful files in Issues, Discussions, or pull requests.

Stay safe and only download FlexClient from official sources.
